nca: Necessary Condition Analysis

nca is a Stata package that implements Necessary Condition Analysis, a method for identifying necessary—but not sufficient—conditions in datasets. It helps researchers uncover constraints that must be present for an outcome to occur.

Installation

Install directly from GitHub using:

 net install nca, from("https://d-spinelli.github.io/nca/") replace

Quick Start 

. use ncaexample, clear 
. nca_analysis individualism innovationperformance, ceilings( ce_fdh ) bottlenecks

This command applies the CE-FDH method and generates a bottleneck plot. Several other methods and plotting options are also available.
